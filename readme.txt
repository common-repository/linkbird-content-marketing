This plugin enables contentbird customers to transfer their created content directly to their wordpress blog and synchronize the content between the contentbird platform and wordpress.

== Description ==

This plugin enables contentbird customers to transfer their created content directly to their wordpress blog. Whenever the wordpress post went online, the plugin will inform the contentbird customer instance and the needed content information like URL or published date will be updated automatically. From that time on contentbird will track the content performance.

== Installation ==

1. Upload all the files to the /wp-content/plugins/ directory or use the Wordpress Admin Panel to install the plugin directly out of the plugin store
2. Activate the plugin in 'Plugins' menu in WordPress
3. Enter your personal installation token on the settings page of the plugin which you will get in your contentbird project wizard
